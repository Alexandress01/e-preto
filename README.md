# É Preto

Plataforma educacional desenvolvida como Trabalho de Conclusão de Curso (TCC) por estudantes do Centro Estadual de Educação Profissional em Tecnologia, Informação e Comunicação (CEEPTIC) de Lauro de Freitas BA.

## Objetivo

Promover a valorização e exploração das histórias, culturas, conhecimentos e contribuições dos povos negros, africanos e originários do Brasil por meio da tecnologia educacional.

## Funcionalidades

- Plataforma educacional online
- Museu Virtual 360°
- Conteúdos históricos e culturais
- Materiais de apoio para estudo
- Interface acessível e responsiva

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- Visual Studio Code

## Impacto

O projeto foi desenvolvido com base em pesquisa realizada com 477 estudantes, identificando lacunas no ensino da História e Cultura Afro-Brasileira, Africana e Indígena.
